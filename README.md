# Haryp2309 Linux-tap

## How do I install these formulae?

`brew install haryp2309/linux-tap/<formula>`

Or `brew tap haryp2309/linux-tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "haryp2309/linux-tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
